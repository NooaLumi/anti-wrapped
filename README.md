# Anti-Wrapped

Share music you like the way you like it, without compromising your privacy. 

Create music cover art you can share online. Enter an artist, a song or album title, and an ekphrasis. Then shape the wave background, play with the colors and download the resulting 1080 x 1920 image you can share anywhere.

**Live site:** <https://antiwrapped.github.io/>

## Features

- WebGL generative wave background with real-time controls
- Optional album art overlay
- Exports a PNG at 1080 x 1920
- Mobile-first, dark-theme UI
- Vanilla TypeScript + Vite

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
